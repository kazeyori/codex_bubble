Codex 额度悬浮球使用说明
========================

一、安装和启动

从 GitHub Release 下载：

codex-bubble-setup-v0.1.4.exe

双击安装器完成安装。安装器会把程序安装到当前用户目录，并创建桌面和开始菜单快捷方式：

Codex 额度悬浮球

以后通过这个快捷方式启动即可。

如果你是在源码或安装目录中手动启动，也可以双击：

启动悬浮球.bat

它会同时启动后台额度同步器和桌面悬浮球。

重复双击不会多开；已经运行时，新启动的悬浮球和后台同步器会自动退出。

如果只单独启动悬浮球，悬浮球也会自动确保后台同步器在运行。

二、如果显示“未连接”

额度和重置时间会显示为“-”，这是正常的未连接占位。

先在这台电脑上使用 Codex 发一条消息，然后等待一分钟。

也可以手动运行：

scripts/run_codex_local_usage_once.bat

三、开机启动

右键运行：

scripts/install_local_usage_startup.ps1

取消开机启动：

scripts/uninstall_local_usage_startup.ps1

四、悬浮球操作

- 左键单击：展开或收起
- 左键拖动：移动位置，位置会自动保存
- 右键单击：打开菜单
- 菜单里可以切换 5小时 / 1周、刷新、检查更新、展开收起、退出

五、在线更新

- 右键悬浮球，选择“检查更新”。
- 如果 GitHub Release 有新版本，软件会打开新版安装器下载地址。
- 下载后直接运行新版安装器即可覆盖升级。
- 安装器会先退出旧版悬浮球和后台同步器，再安装新版。

六、多屏使用

- 可以把悬浮球拖到任意显示器。
- 支持副屏在主屏左侧、右侧或上方。
- 拖到副屏后会保存位置。
- 右键菜单会在鼠标所在屏幕内弹出。

七、卸载

安装目录里有：

卸载悬浮球.bat

也可以在开始菜单点击：

卸载 Codex 额度悬浮球

卸载会退出悬浮球和后台同步器，删除安装目录和快捷方式。
运行时配置、日志和额度缓存暂时保留在 %LOCALAPPDATA%\CodexBubble，方便后续重装继续使用原配置。

八、文件位置

- 默认安装目录：%LOCALAPPDATA%\Programs\CodexBubble
- src/codex_bubble/floating_info_ball.py：悬浮球主程序
- src/codex_bubble/codex_usage_fetcher.py：读取本机 Codex 会话快照
- src/codex_bubble/codex_usage_daemon.py：后台同步器
- config/floating_info_ball_config.json：悬浮球配置
- data/codex_usage_data.json：本机运行时生成，不在分享包中预置
- logs/：运行日志

如果安装目录不可写，配置、额度数据和日志会自动保存到：

%LOCALAPPDATA%\CodexBubble

九、安全说明

当前版本只读取本机 Codex 会话快照中的 rate_limits 字段。
它不读取 cookie、token、auth.json、密码或浏览器登录状态。
它也不会关闭、重启或控制 Codex 程序。
