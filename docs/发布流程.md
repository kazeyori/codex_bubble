# 发布流程

本项目用 Git tag + GitHub Release 做版本发布。

## 版本号规则

使用语义化版本号：

- `0.1.0`：第一个公开预览版。
- `0.x.0`：新增功能或明显体验改进。
- `0.x.y`：问题修复、文档修正或小范围兼容改动。
- `1.0.0`：功能和安装体验稳定后发布正式版。

## 发布前检查

1. 更新 `VERSION`。
2. 更新 `CHANGELOG.md`。
3. 运行语法检查：

```powershell
python -m py_compile src\codex_bubble\floating_info_ball.py src\codex_bubble\codex_usage_fetcher.py src\codex_bubble\codex_usage_daemon.py
```

4. 重新生成 `releases/codex-bubble-vX.Y.Z.zip`。
5. 确认发布包不包含：

- `.git/`
- `data/`
- `logs/`
- `__pycache__/`
- `*.log`
- 本机生成的 `codex_usage_data.json`

## GitHub Release 内容

Release 标题格式：

```text
Codex 额度悬浮球 vX.Y.Z
```

Release 附件：

```text
codex-bubble-vX.Y.Z.zip
```

Release 说明从 `CHANGELOG.md` 对应版本复制。
