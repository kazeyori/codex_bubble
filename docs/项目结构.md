# 项目结构

本项目现在按“用户入口、源码、配置、文档、发布包”拆分。

- 根目录：保留 `README.md`、`AGENTS.md` 和 `启动悬浮球.bat`。
- `src/codex_bubble/`：悬浮球、额度读取器、后台同步器源码。
- `config/`：悬浮球默认配置。
- `scripts/`：英文维护脚本。
- `docs/`：中文使用说明、安全说明和历史方案记录。
- `releases/`：可分享压缩包。
- `data/`、`logs/`：运行时生成，本地保留，不提交。

多屏相关逻辑集中在 `src/codex_bubble/floating_info_ball.py`，使用 Windows 显示器工作区来处理窗口边界、负坐标副屏和右键菜单位置。
